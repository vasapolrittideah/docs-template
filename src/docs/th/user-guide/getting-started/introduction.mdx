# บทนำ

docs-template คือเทมเพลตเว็บไซต์เอกสารที่สร้างด้วย Next.js, MDX และ Tailwind CSS เขียนเอกสารในรูปแบบ MDX ตั้งค่าการนำทางด้วยไฟล์ JSON เพียงไฟล์เดียว และรับเว็บไซต์เอกสารที่มีคุณสมบัติครบครัน ทั้ง Syntax Highlighting, การค้นหาแบบ Full-text, สารบัญ, โหมดมืด, การรองรับหลายภาษา และอื่นๆ อีกมากมาย

<Callout type="info">
สำหรับใช้งานภายในเท่านั้น

เทมเพลตนี้ดูแลรักษาเฉพาะสำหรับโปรเจกต์ภายในของ [TCC Technology](https://www.tcc-technology.com/) และไม่ได้มีไว้สำหรับการใช้งานสาธารณะหรือบุคคลภายนอก

## คุณสมบัติ

คุณสมบัติต่อไปนี้รวมอยู่ในเทมเพลตโดยไม่ต้องตั้งค่าเพิ่มเติม:

- **เนื้อหาบน MDX** — เขียนเอกสารในรูปแบบ Markdown พร้อมรองรับ JSX Component
- **การรองรับหลายภาษา (i18n)** — รองรับหลาย Locale ด้วย [next-intl](https://next-intl.dev/) พร้อมเนื้อหา MDX และการนำทางแยกตาม Locale
- **File-based Routing** — หน้าเพจตรงกับ `src/docs/[locale]/[group]/[slug].mdx`
- **การนำทางด้วย JSON** — กำหนดโครงสร้าง Sidebar ใน `src/docs/[locale]/navigation.json`
- **Syntax Highlighting** — ขับเคลื่อนด้วย [Shiki](https://shiki.style/) พร้อมปุ่มคัดลอกโค้ด
- **การค้นหาแบบ Full-text** — การค้นหาแบบ Fuzzy ฝั่ง Client ด้วย [Fuse.js](https://www.fusejs.io/)
- **สารบัญ** — สร้างอัตโนมัติจาก Heading ในแต่ละหน้า MDX
- **โหมดมืด / สว่าง** — สลับธีมตามค่าที่กำหนดของระบบ
- **Git Metadata** — วันที่แก้ไขล่าสุดและผู้เขียนดึงจากประวัติ Git
- **Callout** — บล็อกแสดงข้อมูล คำเตือน และอันตราย
- **Layout ที่ตอบสนอง** — Sidebar แบบ Drawer สำหรับมือถือ และ Sidebar แบบ Sticky สำหรับเดสก์ท็อป

## โครงสร้างโปรเจกต์

Source Code จัดอยู่ภายใต้ `src/` ดังนี้:

```text
src/
├── app/                        # Next.js App Router pages และ layouts
│   └── [locale]/
│       └── docs/
│           └── [group]/[slug]/ # Dynamic doc page renderer
├── components/
│   ├── common/                 # UI Primitives ที่ใช้ร่วมกัน
│   ├── docs/                   # Components เฉพาะสำหรับ Docs
│   ├── layout/                 # Header, Sidebar, Footer, TOC
│   └── mdx/                    # MDX Component Overrides
├── docs/
│   ├── en/
│   │   ├── navigation.json     # ตั้งค่าการนำทาง Sidebar ภาษาอังกฤษ
│   │   └── [group]/[slug].mdx  # เนื้อหาเอกสารภาษาอังกฤษ
│   └── th/
│       ├── navigation.json     # ตั้งค่าการนำทาง Sidebar ภาษาไทย
│       └── [group]/[slug].mdx  # เนื้อหาเอกสารภาษาไทย
├── i18n/                       # Locale Routing และตั้งค่า next-intl
├── messages/                   # UI Strings (en.json, th.json)
├── lib/                        # Utilities สำหรับ MDX, Git และการค้นหา
└── proxy.ts                    # Middleware ของ next-intl สำหรับ Locale Routing
```
